🧠 1. Kernel (Nhân hệ điều hành)
Đây là trung tâm của hệ điều hành giúp giao tiếp giữa phần cứng và phần mềm.
Kernel có nhiệm vụ chính:
🏗 Quản lý tiến trình (Process Management): Quản lý các chương trình đang chạy.
💾 Quản lý bộ nhớ (Memory Management): Cấp phát và giải phóng bộ nhớ.
📂 Quản lý hệ thống tập tin (File System Management): Điều khiển quyền truy cập và lưu trữ dữ liệu.
🌐 Quản lý mạng (Networking): Giao tiếp giữa các thiết bị trong và ngoài hệ thống.
🔌 Quản lý thiết bị (Device Management): Tương tác với phần cứng thông qua driver.

-----------------------------------------------------------------------------------------------------------------

💻 2. Shell (Giao diện dòng lệnh)
Shell là một chương trình giúp người dùng tương tác với hệ điều hành.
Có thể nhập lệnh (commands) để thực hiện các thao tác như tạo file, chạy chương trình, quản lý tiến trình, v.v.
Một số loại Shell phổ biến:
🐧 Bash (Bourne Again Shell) – mặc định trong nhiều hệ thống Linux.
🛠 Zsh (Z Shell) – giống Bash nhưng có nhiều tính năng mạnh hơn.
🔷 Fish (Friendly Interactive Shell) – đơn giản, dễ dùng.

-----------------------------------------------------------------------------------------------------------------

🛠 3. Utilities (Công cụ & tiện ích)
Bao gồm các chương trình hỗ trợ quản lý hệ thống như:
🗂 Quản lý file: ls, cp, mv, rm
⚙ Quản lý tiến trình: ps, top, kill
🌐 Quản lý mạng: ping, ifconfig, netstat
🔑 Quản lý người dùng: who, passwd, chmod

-----------------------------------------------------------------------------------------------------------------

🏗 4. Applications (Ứng dụng)
Các chương trình chạy trên Linux như trình duyệt web (Firefox, Chrome), bộ soạn thảo (VS Code, Vim), máy chủ web (Apache, Nginx), cơ sở dữ liệu (MySQL, PostgreSQL), v.v.
Ứng dụng có thể chạy dưới dạng GUI (giao diện đồ họa) hoặc CLI (giao diện dòng lệnh).
